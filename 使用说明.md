# 抖音爬虫使用说明

## 1. 环境准备

```bash
pip install -r requirements.txt
playwright install chromium
```

## 2. 启动 MongoDB

确保 MongoDB 运行在 `localhost:27017`，已开启认证（`config/settings.yaml` 中配置用户名密码）。

## 3. 首次登录（必须）

抖音搜索需要登录态。运行以下命令，在弹出的浏览器窗口中扫码登录：

```bash
python scripts/login.py
```

登录成功后回到终端按 Enter，Cookie 保存到 `config/browser_state.json`。后续爬虫会自动复用，无需再次登录。

## 4. 运行爬虫

### Feed 模式 — 首页推荐流，无差别爬取

```bash
python scripts/run_scraper.py --mode feed --max-videos 20
```

无需指定关键词，直接爬取抖音首页推荐流的视频。

### 搜索模式 — 按关键词搜索爬取

```bash
# 使用 config/keywords.yaml 中的关键词
python scripts/run_scraper.py

# 命令行指定关键词
python scripts/run_scraper.py --keywords "AI,大模型,科技" --max-videos 15

# 不爬评论
python scripts/run_scraper.py --keywords "人工智能" --no-comments
```

关键词默认从 `config/keywords.yaml` 读取。

### 调试

```bash
python scripts/run_scraper.py --headful --debug --max-videos 3
```

## 5. 数据存储

爬取数据自动写入 MongoDB `douyin_research` 数据库：

| 集合 | 内容 |
|------|------|
| `videos` | 视频元数据（描述、作者、统计、话题标签） |
| `comments` | 评论（正文、用户、IP 属地、发布时间） |

## 6. 命令行参数

| 参数 | 说明 |
|------|------|
| `--mode` | `search`（关键词搜索）或 `feed`（首页推荐），默认 `search` |
| `--keywords` | 逗号分隔的搜索关键词 |
| `--max-videos` | 最多爬取的视频数 |
| `--max-comments` | 每个视频最多爬取的评论数 |
| `--no-comments` | 跳过评论爬取 |
| `--headful` | 显示浏览器窗口（默认无头模式） |
| `--debug` | 开启 DEBUG 级别日志 |
